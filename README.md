## Installation

```bash
$ yarn install
```

## Migration up

```bash
$ yarn migrate:up
```

## Migration down

```bash
$ yarn migrate:down
```

## Running the app

```bash
# development
$ yarn run start

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
